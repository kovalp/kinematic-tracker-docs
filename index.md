# Kinematic tracker

Welcome to kinematic tracker documentation.

The latest version of the `kinematick-tracker` is `16.1.0`.

The package `kinematic-tracker` provides a number of algorithms for multiple-object tracking (MOT).
MOT is also often referred as multiple-target tracking (MTT).
